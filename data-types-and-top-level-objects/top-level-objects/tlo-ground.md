# TLO:Ground

## Description

Object which references the ground spawn item you have targeted.

## Forms

* [_ground_](../data-types/datatype-ground.md) **Ground**

## Access to Types

* [_ground_](../data-types/datatype-ground.md) **ground**

## Examples

`/echo ${Ground.Distance}`

Echos the distance to the ground item you have targeted.
