# Clearerrors

## Syntax

**/clearerrors**

## Description

Clears each of the last errors in the [_macroquest_](../../data-types-and-top-level-objects/data-types/datatype-macroquest.md) type:

* **${MacroQuest.Error}**
* **${MacroQuest.MQ2DataError}**
* **${MacroQuest.SyntaxError}**

## Example

\(need before and after\)
