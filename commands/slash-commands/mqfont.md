# Mqfont

## Syntax

**/mqfont \#**

## Description

Changes the size of the font in the MQ2ChatWnd.

**Notes**

* The font sizes are not the same as EQ's chat window sizes
* Suggested use: numbers in the range of _-3_ to _2_
