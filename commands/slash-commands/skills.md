# Skills

## Syntax

**/skills \[** _**skillname**_ **\]**

## Description

Lists the skill level\(s\) that match _skillname_ if provided, or all possible skills if no parameter given.

## Examples

* Lists all of your skills with skill level

  ```text
  /skills
  ```

* Returns your pottery skill

  ```text
  /skills pottery
  ```
