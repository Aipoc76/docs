# Altkey

## Syntax

**/altkey** _**command**_

## Description

Execute _command_ while telling the window manager that the alt key is pressed. Can also be used together with [/ctrlkey](ctrlkey.md) , [/shiftkey](shiftkey.md) , or both as in:

```text
 /ctrlkey /altkey /shiftkey command
```
