# TLO:DoorTarget

## Description

Object used to return information on your doortarget.

## Forms

* [_spawn_](../data-types/datatype-spawn.md) **DoorTarget**

## Access to Types

* [_spawn_](../data-types/datatype-spawn.md) **spawn**

## Examples

```text
/echo ${DoorTarget.ID}
```

Returns the ID of your door target.
