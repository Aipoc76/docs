## Description

Creates an object to access your friends list

## Forms

-   *[friends](../data-types/datatype-friends.md)* **Friends**

## Access to Types

-   *[friends](../data-types/datatype-friends.md)* **friends**

## Examples

`  /echo ${Friends}`

Echos the number of friends you have

`  /echo ${Friends.Friend[1]}`

Echos the name of your first friend
