## Description

## Forms

-   *[macroquest](../data-types/datatype-macroquest.md)* **MacroQuest**

## Access to Types

-   *[macroquest](../data-types/datatype-macroquest.md)* **macroquest**

## Examples

`  /echo ${MacroQuest.LastTell}`

Returns the name of the last person that sent you a tell.
