## Description

A time object indicating EQ Game Time.

## Forms

-   *[time](../data-types/datatype-time.md)* **GameTime**

## Access to Type

-   *[time](../data-types/datatype-time.md)* **time**

## Examples

`  /echo ${GameTime.Date}`

Echos todays in-game date to the chat window.
