## Description

Object used to return information on your doortarget.

## Forms

-   *[spawn](../data-types/datatype-spawn.md)* **DoorTarget**

## Access to Types

-   *[spawn](../data-types/datatype-spawn.md)* **spawn**

## Examples

    /echo ${DoorTarget.ID}

Returns the ID of your door target.
